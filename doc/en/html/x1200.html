4.2. The Parameter File

Generic file name: base.xml

Format: XML

The parameter file is a generic file which contains information required by many related programs; NeuroScope only needs a subset of this information. The parameter file is optional: it is usually generated using NDManager, a dedicated application, but if you do not provide it, NeuroScope will prompt you for the necessary information and automatically generate a parameter file for you (however, contrary to NDManager, NeuroScope will actually generate only the subset of fields that it requires, and not a full parameter file).

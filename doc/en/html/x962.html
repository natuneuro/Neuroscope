2.11. File Properties

Selecting File->Properties brings up the File Properties dialog. In this dialog, you can check and edit a number of parameters related to the current session. These parameters are described in Settings (notice that although the File Properties dialog is very similar to the Defaults tab of the Settings dialog, the former is related to the current session only, whereas the latter defines the default values for new sessions). Changes in the File Properties dialog will be applied immediately upon validation.

One item in the File Properties dialog needs special mention: the Sampling Rate for this File entry. This is useful when you work with data files other than the .dat file (which contains raw data, and thus has the same sampling rate as the acquisition system).
